1. 打开环境 URL，用 `seele` 登录  
2. 进入工作区 → **新建空画布项目**（不要用旧项目，避免脏数据）  
3. 进入画布，确认基本为空  

**记录**：项目名 / URL：`________________________`

### 0.2 浏览器

- Chrome / Edge  
- F12 → **Network** → 勾选 **Preserve log**  
- 常用过滤：`agent/run`、`short-video`、`build-shot`、`model-gateway`

### 0.3 Agent 模式

打开 Agent 面板，本手册默认用：

| 模式 | 本手册 |
|------|--------|
| **手动确认** | **必测**（每步看清再「采纳并继续」） |
| 自动生成 | 可选，易跳步 |

### 0.4 不要误报为 Bug 的情况

| 现象 | 说明 |
|------|------|
| 第一次先出创意方案卡 | 正常，选一个再继续 |
| `split_shot_beats`（拆节拍）被跳过 | 可选步，允许 |
| 文本/视频生成转圈 1–3 分钟 | 等完成再点继续 |
| 短视频 `visual_source=stock` 无 Pexels Key | 会回退静图幻灯，属设计 |
| Cloudflare 1033 / 整站打不开 | 运维问题，报主程，不是功能 Bug |

# A. OpenMontage 相关（Agent 编排）

**白话**：Agent 按固定步骤走剧本链路；左边有流水线面板；「技能」可以点。

## A1. Agent 主链前三步（必测）

**操作**

1. Agent 设为 **手动确认**  
2. 发送：

```text
帮我做一个 30 秒品牌宣传片，主题是重庆火锅，要有故事感，先给几个方向方案
```

3. 若出创意卡 → 选一个方案  
4. 逐步点 **采纳并继续** 或发「继续」，依次验收：

| 步 | 期望（Network 或画布） | □ |
|----|------------------------|---|
| 1 | 出现文本输入卡；`pipeline_step` ≈ `create_text_note` | |
| 2 | 剧本文本生成完成；`start_text_generation` | |
| 3 | 出现大纲节点；`generate_outline` | |

**Network**：过滤 `agent/run`，Response 里搜 `pipeline_step`。

| 结果 | □ PASS　□ FAIL | 备注：________ |

# B. DramaClaw 相关（角色一致性）

**白话**：角色要有身份和参考图；绑了角色却没配齐时，出图/出视频应被拦住。

## B1. 完善角色资产（必测）

**操作**

1. 用 Agent 或手动，在分镜链路里做出 / 打开 **角色设定**（cast / 角色卡）  
2. 给某个角色填写或上传：  
   - 身份 ID（如 `alice_default`，界面字段可能叫 identity）  
   - 至少一张参考图（脸 / 三视图 / 服装 / 主图任一）  
3. 在 **分镜表** 某几行绑定该角色（identity 绑定或 @ 角色，以界面为准）  

**期望**

- [ ] 角色能保存参考图  
- [ ] 分镜行能关联到该角色  

| 结果 | □ PASS　□ FAIL | 备注：________ |

# C. Pixelle + MoneyPrinter 相关（短视频工厂）

**白话**：主题一键生成短视频；可配音对字幕；可选素材；画布上有专门节点。

## C1. 画布「短视频工厂」节点（必测）

**添加方式（任选一种）**

- 左侧工具栏 **「+」** → **「短视频工厂」**  
- 或 **双击空白画布**  
- 或 **右键菜单** → **「短视频工厂」**  

**操作**

1. 添加节点后填写：  
   - 主题：如 `城市夜景短视频`  
   - 段数：3  
   - 打开配音（enable_tts）  
   - 打开烧字幕（若有）  
   - 画面来源：先选 **幻灯 / slide**（稳）  
2. 点 **生成**，等待完成  
3. 节点内应能预览或打开成片  

**期望**

- [ ] 能添加节点  
- [ ] 任务从排队 → 完成（允许 1–5 分钟）  
- [ ] 能看到成片预览/下载  

| 结果 | □ PASS　□ FAIL | 备注：________ |

## C2. 配音与字幕时间轴（必测，可用 API）

> 若画布节点不好观察字幕时间，可用本机/内网 API（问主程要 Token）。

```bash
# 登录拿 TOKEN 后：
curl -s -X POST http://127.0.0.1:7788/api/short-video/generate \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "topic": "人工智能如何改变生活",
    "segment_count": 3,
    "burn_captions": true,
    "enable_tts": true,
    "voice_name": "zh-CN-XiaoxiaoNeural",
    "visual_source": "slide"
  }'
```

再轮询：

```bash
curl -s http://127.0.0.1:7788/api/short-video/{task_id} \
  -H "Authorization: Bearer $TOKEN"
```

下载成片（可选）：

```bash
curl -s http://127.0.0.1:7788/api/short-video/{task_id}/file \
  -H "Authorization: Bearer $TOKEN" -o final.mp4
```

**期望**

- [ ] `status` 最终为完成类状态  
- [ ] 成片有人声（真 TTS 环境）或至少段长 **不是** 机械的每段整 2 秒（有配音时）  
- [ ] 打开字幕时，字幕大致跟旁白走（主观即可）  

> 若环境无外网 Edge TTS：主程可能开了 MOCK；成片可能无真声音，但任务应成功。问清再测，不要误判 FAIL。
